How it works

TrueTrend AI reads live market data, computes technical context (trend, momentum, levels), and returns a clean bias (long/short/neutral) with confidence. It logs “why you won/lost” so you improve faster. No copy-trading—this is your edge, not someone else’s.

Connecting Alexa (Pro)

  1. In your Amazon Developer Console, create a Custom Alexa Skill named TrueTrend.
  2. Set the HTTPS endpoint to: https://YOUR-DOMAIN/api/alexa
  3. Add intent GetDirectionIntent with a slot symbol (type: plain text).
  4. Sample utterances: “what’s the direction on {symbol}”, “ask TrueTrend about {symbol}”.

That’s it. Try: “Alexa, ask TrueTrend what’s the direction on NVDA?”

Strategies (built-in)

  • Trend-SMA 20/50 — Baseline engine (20/50 SMA slope/cross + RSI filter).
  • Break of Structure — Higher-highs/lower-lows + volume to catch regime changes.
  • Support/Resistance — Auto key levels from recent swing highs/lows (ATR buffers).
  • ORB (Opening Range Breakout) — Session range + momentum to catch early trend days.
  • Portfolio Mode — Sector trends & ETF rotation (momentum + breadth).
  • Options Helpers — Directional bias + IV context to plan debit/credit structures.

Combine or customize these—TrueTrend explains each decision in your journal so you keep refining.

Charts & Timeframes

TrueTrend works with TradingView and most broker charts (equities, ETFs, futures, forex, crypto). Use any timeframe: 1m, 5m, 15m, 30m, 1h, 4h, Daily, Weekly, Monthly.

How to use it

  • Type a symbol in the demo (NVDA, AAPL, GLD, EUR/USD, BTC/USD) and choose a timeframe.
  • TrueTrend returns bias + confidence + RSI + key levels and a short rationale.
  • In the app, your journal stores “why you won/lost” so you improve quickly.

Tips

  • Match your chart timeframe to the one you ask TrueTrend for.
  • Combine bias with your risk rules (e.g., stop/target using recent S/R).
